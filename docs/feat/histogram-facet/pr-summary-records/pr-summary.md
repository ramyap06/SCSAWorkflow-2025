## Summary

This PR completes the histogram facet work by adding a reviewer-ready facet path to `spac.visualization.histogram()` and aligning the template boundary with that API. Faceted histograms now support shared bins, grouped count-table returns, figure-level labels, grouped-mode guardrails, adaptive layout for long categorical labels, and template-side facet controls that stay inactive outside facet/grouped modes.

## Changes

- Core histogram behavior
  - Adds `facet=True` support in `histogram()` for grouped plots, with guardrails that require `group_by` and disallow `together=True`.
  - Adds `_derive_facet_geometry()` to derive facet layout from group count, figure-size hints, and rotated long-label burden.
  - Reuses shared grouped histogram tables for grouped-overlay and facet paths so numeric and categorical facets stay aligned and the returned `df` follows one grouped count-table pattern.
  - Normalizes default-like `bins` inputs to the Rice-rule fallback for numeric histograms, while categorical grouped/facet paths preserve category slots and ignore numeric-style `bins` changes.
  - Uses figure-level axis labels in facet mode and blocks unsupported external-`ax` usage for grouped-separate and facet layouts.

- Template boundary
  - Exposes and validates `Facet`, `Facet_Ncol`, and `Max_Groups` in `run_from_json()`.
  - Treats `Figure_Width` / `Figure_Height` as `"auto"` defaults in facet mode, forwarding explicit size hints only when both are provided.
  - Keeps seaborn-native `element` and `stat` controls lightly normalized, forwards `multiple` only for grouped same-axis overlays, and ignores grouped-only or facet-only hints when the corresponding mode is off.
  - Adjusts facet titles and layout handling so faceted figure titles and rotated tick labels remain readable across denser grids.

- Tests and docs
  - Adds dedicated unit coverage for `_derive_facet_geometry()`.
  - Expands histogram tests for facet smoke behavior, shared-bin consistency, long-label sizing, bins fallback, facet-size hints, `max_groups`, categorical facet bin handling, and external-`ax` guardrails.
  - Updates histogram/template docstrings and local PR tracking docs to match the final facet contract.
